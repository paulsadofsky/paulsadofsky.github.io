LED Wall

IDEA Lab: Undergraduate Research Intern

Project Description

_____________________

This project's aim was to update and improve the LED matrix wall display located in the basement of Benedum Hall in Pittsburgh, PA. This wall had originally been designed and constructed several years back and had severe issues that inhibited its ability to be functional.

Project Goals

_____________________

The goals of this project included stabilizing the software ran to operate the LED matrix, adjust hardware issues that caused color inaccuracies, and add addtional hardware to implement interactive features to the display. The project lasted for approximately 8 weeks.

Group Members

_____________________

This project was entirely executed by myself, with supervision and oversight from my supervisors.

Results

Due to the severity of the issues that plagued the design, it was determined that instead of attempting to individually diagnose and find solutions to each problem, it was significantly more efficient and sustainable for the entire LED matrix to be re-engineered and reconstructed from the ground up. Therefore, the deliverable of the project was a 20-page proposal for an entire redesign of the wall that is awaiting approval and funding, as well as OpenCV hand tracking on the NVIDIA Jetson Nano to be integrated into the wall upon the completion of the redesign.
