Budgeting Database and Website

Great Neck Park District

Project Description

_____________________

This project's aim was to entirely overhaul the outdated budgetting database and corresponding website that the Great Neck Park District, located in Great Neck, NY, used to track all of their expenses from projects and events for the fiscal year.

Project Goals

_____________________

The project resulted in a fully reconstructed website and integrated database using the Flask and SQLAlchemy platforms, along with LDAP interfacing and over 25 pages of user documentation.

Group Members

_____________________

This project was entirely executed myself, with supervision and oversight from my employer. I fully designed and implemented the back-end, front-end, database and documentation.

Results

I drove a 200% improvement in the database load time and increased the compatibility to 5x the number of operating systems & browsers by identifying the ideal tech stack to support the district's needs. Additionally, I developed bespoke developer features that allowed staff to access manual database adjustments and backup & restore the system.
