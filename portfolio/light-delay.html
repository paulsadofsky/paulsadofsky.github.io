Minecraft 3-bit Multiplier

Personal Project

Project Description

_____________________

This project's aim was to conceptualize the idea of digital logic circuits to further my personal understanding of digital logic, and also as part of a larger presentation to educate grade school students on the inner workings of digital logic through an accessible medium.

Project Goals

_____________________

The goal was to successfully implement a 3-bit by 3-bit digital circuit multiplier within the popular video game Minecraft using the redstone system, which functions like electricity.

Group Members

_____________________

This project was entirely designed and executed by myself.

Results

This project resulted in a successful implementation of the 3-bit by 3-bit multiplier, alongside a 10-page documentation detailing the specifics of the implementation.
